# ContentLite (DK)

Letvægts PHP-baseret indholdsstyring med nyheder, galleri, adminpanel og førstegangsinstaller.

## Indhold

- Nyhedssektion med TinyMCE-baseret redigering
- Galleri med albummer og billeder
- Førstegangsinstaller til databaseopsætning
- Understøttelse af tabel-prefix i delt database

## Opsætning

1. Upload filerne til serveren.
2. Åbn `install.php` i browseren.
3. Udfyld databaseoplysninger og eventuelt tabel-prefix.
4. `install.php` låses automatisk efter installation.
5. Ret CSS-selektorenes parametre selv efter krav/behov i.f.m. layout
